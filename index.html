To help with Try Transit Week, Code for America has created this site in partnership with the Macon Transit Authority.

We're also using GPS data to create an upcoming real-time SMS text-message application that would allow users to see when the nearest bus is coming, along with an online trip-planning tool for MTA riders. A mobile iPhone application is also in the works.

Check back here for periodic updates, and let us know what tools you think would make transit easier by sending us an email.

As you're an iPhone user, make sure to add this site to your home-screen, too.
