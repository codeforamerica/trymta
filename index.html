ABOUT CFA

Code for America is a non-profit based in San Francisco. Each year, we partner developers and designers from our fellowship program with cities to improve the way citizens interface with local government.

The Team

Macon, Georgia is one of the 8 cities Code for America is working with in 2012. Macon's CfA team members are: Zach, a developer from Fort Worth; Nick, a digital mapper from Nasuha, NH; and Jessica, an urban designer born and raised in nearby Warner Robins, Ga.

INTERACTIVE SYSTEM MAP

We've created a new and interactive web map of Macon's transit system. The map shows all of the active routes in the system as well as the new BIRD (Bus into Robins Daily) route to Robins Air Force Base.

Each route also includes bus stops so that users can find the stops nearest them. When your mouse hovers over a stop, details for the stop are revealed.

You can also view the map on the MTA's website.

TEXT-FOR-BUS APP

Using historic data gathered from the GPS systems on each of the MTA's buses, we've created an application that allows anyone with a text-enabled phone to get information about bus schedules.

How to

  • Text your location to 487-426-7272. No need to include the city; write an address like this:

    123 Main Street

  • The system will text you back the stop nearest to you as well as the next bus to come inbound (towards Terminal Station) or outbound (away from Terminal Station) to that stop.
  • If you need help, just text "help".

Give it a try!

No smart phone necessary and nothing to download - if you can text, you can get bus times!

SPLOST

CfA's Macon team is also working on a website to visualize the budget of the general SPLOST that was passed earlier this year. At the website, residents will be able to see exactly where projects are located, how much money a project will recieve and when its funds are disperesed.
